# 📊 Stock Market Analyzer - Team Project

### Overview
Our team is building this Stock Market Analyzer to solve a simple problem: making sense of huge amounts of historical stock data. Instead of just looking at daily prices, we are using Python to calculate key trends like Moving Averages and Market Volatility.The goal of this project is to create a clear flow—starting from fetching raw data via Yahoo Finance, cleaning it to remove errors, and finally visualizing it through charts that show where the market might be heading. We are currently focusing on comparing different sectors to see which stocks are performing better than the overall market index.

### Tech Stack
-  **Python (Data Analysis):** Using Pandas and NumPy for data cleaning, calculating Moving Averages, and Volatility.
-  **SQL (Data Management):** Querying historical stock databases to extract specific trends and performance metrics.
-  **Power BI & Tableau (Visual Dashboards):** Creating interactive and real-time dashboards for stock performance tracking.
-  **MS Excel (Initial Analysis):** Handling exploratory data analysis (EDA) and quick statistical summaries.
-  **Data Source:** Yahoo Finance / Historical CSV

### Folder Structure
- `/data`: Raw and processed stock datasets.
- `/notebooks`: Python code for analysis.
- `/visuals`: Graphs, charts, and dashboards.
