# 📊 Stock Market Analyzer - Team Project

### Overview
This repository contains a comprehensive data analysis of stock market trends. We focus on historical price movements, technical indicators, and volatility analysis.

### Tech Stack
-  **Python (Data Analysis):** Using Pandas and NumPy for data cleaning, calculating Moving Averages, and Volatility.
-  **SQL (Data Management):** Querying historical stock databases to extract specific trends and performance metrics.
-  **Power BI & Tableau (Visual Dashboards):** Creating interactive and real-time dashboards for stock performance tracking.
-  **MS Excel (Initial Analysis):** Handling exploratory data analysis (EDA) and quick statistical summaries.
-  **Data Source:** Yahoo Finance / Historical CSV

### Folder Structure
- `/data`: Raw and processed stock datasets.
- `/notebooks`: Python code for analysis.
- `/visuals`: Graphs, charts, and dashboards.
